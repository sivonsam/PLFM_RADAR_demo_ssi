Overview

The Athene demo showcases a Maritime Situational Awareness (MSA) platform built on the AERIS-10 open-source phased-array radar (10.5 GHz, up to 20 km range), fused with live and simulated data sources and presented through an ESRI ArcGIS + Microsoft Azure cloud stack. The demo area is the Gulf of Finland / Tallinn–Uurainen corridor — a strategically relevant, data-rich maritime environment.

2 Azure Platform Foundation

Microsoft Fabric · OneLake · Eventhouse · AI Foundry as data backbone.

Core storage

Microsoft Fabric + OneLake

Unified analytics workspace. Single Delta/Parquet lake shared across Fabric workloads and exportable to ArcGIS. Lakehouses per domain: radar, AIS, weather, environment.

Streaming

Azure Eventhouse (KQL)

Streaming ingestion from Athene simulator & ESRI Velocity. KQL queries for track analytics and time-series anomaly detection. Fabric Real-Time Intelligence.

Message bus

Azure Event Hub

Bridges Athene simulator → Eventhouse → downstream consumers. Partitioned by radar site for parallel ingestion.

AI hub

Azure AI Foundry

Central model deployment, prompt flows, and agent orchestration. Connects to Fabric data. Hosts custom ML models for vessel classification and anomaly detection.

Batch ingestion

Fabric Pipelines

Batch ingestion of open data (AIS snapshots, FMI, EMODnet) into OneLake. Scheduled refreshes keep reference data current.

Identity & access

Microsoft Entra ID

Role groups map to OneLake row-level security + ArcGIS item sharing. Single sign-on across Fabric, ArcGIS, and Copilot Studio agents.

Key AI Demo Scenarios

1

Dark Target Detection

Radar track exists, no AIS correlation → agent raises alert, queries Copernicus Sentinel-1 SAR imagery, generates incident report automatically.

2

Vessel Behaviour Anomaly

Speed/heading deviation from historical AIS pattern → Foundry IQ flags and explains its reasoning chain to the analyst.

3

Natural Language Tactical Briefing

Johtaja asks Copilot "What is the current threat picture in sector Alpha?" → synthesised answer from radar + AIS + weather in seconds.

4

Developer Playground

Kehittäjä calls Foundry REST APIs to test custom prompt flows against live Eventhouse data and radar streams with GitHub Copilot assistance.
